## 🌟 Описание
Shazam v1.0.2 — обновленная версия мощного модуля для распознавания музыки! В этой версии мы полностью отказались от использования внешних API-токенов. Благодаря интеграции библиотеки **ShazamIO**, модуль стал полностью автономным, более быстрым и стабильным. 🔍🎶

## 🚀 Установка
```bash
.dlm [https://raw.githubusercontent.com/apcecoc/Shazam/main/Shazam.py](https://raw.githubusercontent.com/apcecoc/Shazam/main/Shazam.py)
````

## 🛠 Команды

### 📥 Распознавание музыки

| Команда | Описание |
|---------|----------|
| `.shazam` | Распознаёт музыку из видео/аудио (ответьте на сообщение) |

## 🎨 Особенности

  - ⚡ **Полная автономность** — работа через ShazamIO без сторонних API-ключей.
  - 🚀 **Мгновенный поиск** — оптимизированные запросы напрямую к серверам Shazam.
  - 💾 **Экономия ресурсов** — эффективная обработка медиафайлов через FFmpeg.
  - 🔄 **Smart Cleanup** — мгновенное удаление временных файлов после обработки.

## 🔧 Технические детали (v1.0.2)

  - **Engine:** Прямое взаимодействие с Shazam API через библиотеку `shazamio`.
  - **No Proxy Needed:** Оптимизированные заголовки запросов для обхода ограничений.
  - **FFmpeg Processing:** Нарезка аудио (15 сек) для минимальной задержки.
  - **Stability:** Исключены ошибки, связанные с истечением срока действия токенов.

## 🌟 Description

Shazam v1.0.2 is the latest evolution of the music recognition module\! This update introduces a **Token-Free architecture**. By switching to the **ShazamIO** engine, the module is now completely autonomous, faster, and more reliable than ever. 🔍🎶

## 🚀 Installation

```bash
.dlm [https://raw.githubusercontent.com/apcecoc/Shazam/main/Shazam.py](https://raw.githubusercontent.com/apcecoc/Shazam/main/Shazam.py)
```

## 🛠 Commands

### 📥 Music Recognition

| Command | Description |
|---------|-------------|
| `.shazam` | Recognizes music from video/audio (reply to message) |

## 🎨 Features

  - ⚡ **Full Autonomy** — works via ShazamIO with zero external API tokens required.
  - 🚀 **Instant Search** — optimized direct requests to Shazam servers.
  - 💾 **Resource Efficient** — streamlined media processing using FFmpeg.
  - 🔄 **Smart Cleanup** — temporary files are deleted immediately after use.

## 🔧 Technical Details (v1.0.2)

  - **Engine:** Direct interaction with Shazam API via `shazamio` library.
  - **No Proxy Needed:** Optimized request headers to avoid rate limits.
  - **FFmpeg Processing:** Media conversion limited to 15s for low latency.
  - **Stability:** Fixed all issues related to expired or invalid API keys.
